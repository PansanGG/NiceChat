# NiceChat

## Plugin for maximum chat customization.

Features:
  - Chat formatting
  - Profanity filter
  - Unique messages for groups (LuckPerms)
  - In future will be more!

Requires:
  - Placeholder API for placeholders
  - LuckPerms for groups
  (without these plugins NiceChat won't enable)
